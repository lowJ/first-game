How to play
Install Processing (https://processing.org/download/)
Download the game folder.
Rename the folder so it has the same name as the Game.pde file inside of it(so rename it to "Game").
Open the Game.pde file with processing. Then run it by hitting the button on the top left.

Controls
P1
wasd movement.
shift, lock direction facing
space, shoot

P2
ijkl, movement.
n, lock direction facing
semi colon, shoot

Map Editor
The map can be edited by opening up Game.pde and changing the tiles array. 
The theme of the map can be changed by editing the mapType variable.
